# Simple Interest Calculator

This project is a simple Bash script that calculates simple interest based on user input.

## Project Name

Simple Interest Calculator

## Description

The purpose of this project is to demonstrate basic Git and GitHub workflows, including repository forking, cloning, editing files, committing changes, pushing updates, creating branches, merging branches, and creating pull requests.

The project includes a shell script named `simple-interest.sh` that calculates simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100
